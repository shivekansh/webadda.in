# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-21
### Added
- Initial release of webadda.in marketing website.
- Modern responsive UI with Framer Motion animations.
- Complete set of sections: Hero, Portfolio, Pricing, Process, FAQ, and Contact.
- Production-ready Vite build with aggressive code-splitting.
- Enhanced accessibility (A11y) including keyboard navigation and ARIA roles.
- Hardened security with Content Security Policy (CSP), rate limiting, and safe WhatsApp redirection.
- SEO improvements via meta tags and JSON-LD structured data.
- JavaScript disabled fallback experience.
